## Code for Reproducing Figures
We provide sample code for reproducing the figures presented in our paper entitled `Sobolev-Ricci Curvature`, submitted to ICML 2026, as part of the supplementary material.
The code is organized into two subdirectories corresponding to the experimental settings in the paper: [community detection](./community_detection) and [MANL](./manl).

Detailed descriptions of the implementations, as well as the mapping between figures and code, are provided in the `README.md` files within each subdirectory.
